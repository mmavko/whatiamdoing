whatiamdoing

whatiamdoing is a super-simple personal time tracker.

Usage

Just write down the log of your work and let the app calculate the stats. Valid lines get checkmarks.

Categorize the tasks using semicolons. Use the following keywords to insert the timestamps: 'now', 'teraz', 'тепер', 'зараз', 'теперь', 'сейчас'.

If you want to log more than 1 day, then finish every day with one of these tasks: 'home', 'end'.

The project is work in progress. File issues and suggestions here: https://github.com/mmavko/whatiamdoing/issues.
